# GAA

Proyecto web para explorar videojuegos, consolas, periféricos, figuras, ropa y noticias.

## Desarrollo
- Instalar dependencias: `npm install`
- Desarrollo local: `npm run dev`
- Build de producción: `npm run build`

## Datos
- Los archivos JSON ubicados en `public/JSON/` fueron creados mediante scraping usando Python.
- Ejemplos de rutas: `public/JSON/noticias/`, `public/JSON/videojuegos/`, `public/JSON/ropa/`, etc.

## Notas
- Algunas entradas pueden variar en sus campos (`title`, `category`, `image_url`, `redireccion_url`, `price`).
- Las imágenes del carrusel provienen de `public/img/`.
